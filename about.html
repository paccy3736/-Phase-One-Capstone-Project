About Book Explorer

A passion project built to help readers discover their next great read.

What is Book Explorer?

Book Explorer is a responsive, interactive web application that lets you search millions of books, save your favourites, and discover new titles — all powered by the Open Library API. Whether you're looking for a classic novel or the latest bestseller, Book Explorer helps you find it fast.

Features

Search Books

Search by title across millions of books from the Open Library database.

Save Favourites

Add books to your personal favourites list — saved locally so they persist across sessions.

Fully Responsive

Works beautifully on mobile, tablet, and desktop using a mobile-first Tailwind CSS design.

Fast & Lightweight

Built with vanilla JavaScript ES6 modules — no heavy frameworks, just clean and fast code.
